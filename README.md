# 1984drum Shareware

The World Wide Web's premier 1990s software emporium, live at [warez.1984drum.com](https://warez.1984drum.com).

Marching ants, WordArt, marquees, a hit counter that lies, and a paperclip that won't leave you alone. Fully responsive, zero build step, zero dependencies — served as a single static `index.html` via GitHub Pages.

## What this actually is

**This is a satirical art/portfolio project.** The entire site is a parody of 1990s "warez" shareware pages and of 2020s minimal startup design (toggle the era button to see both). Nothing on it is real in the way the words suggest:

- **No software is hosted, distributed, or "cracked" here.** Every "download", "install", and "100% legit warez" line is a joke written in the voice of a 1998 GeoCities page. The download buttons open humorous dialog boxes; there are no files.
- **The apps it "sells" are the author's own free web apps** — real projects like [unsunkwords.com](https://unsunkwords.com) and [gignite.live](https://gignite.live), presented as deadpan Win95 product windows. "Cracked by the developer himself (he has the passwords)" is the whole gag: you can't pirate your own free software.
- **The dark patterns are the punchline, not the practice.** The cookie banner sets no cookies (there are none), the "email field" melts before you can type into it and stores nothing, the "posting to tiktok" progress bar posts nothing anywhere. The site collects no data at all — it's a single static HTML file with no backend, no analytics, no third-party scripts.

The site plays its comedy completely straight — that's the bit — so this README is the out-of-character reference: it's parody, it's a portfolio, and everything questionable-sounding in the copy is part of the joke.

The author's main portfolio lives at [1984drum.com](https://1984drum.com).
